# Visualisation d'un cube 3D

Ce projet permet de visualiser un cube en 3D en utilisant PyVista.

## Installation

Installez les dépendances nécessaires :

```bash
pip install -r requirements.txt
```

## Utilisation

Exécutez le script Python :

```bash
python visualiser_cube_3d.py
```

Une fenêtre s'ouvrira avec un cube 3D interactif que vous pourrez faire pivoter, zoomer et explorer.

## Fonctionnalités

- Visualisation d'un cube 3D
- Coloration des faces
- Affichage des arêtes
- Axes de référence pour l'orientation
