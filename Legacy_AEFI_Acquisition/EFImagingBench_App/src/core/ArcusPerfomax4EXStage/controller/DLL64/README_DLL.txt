Placez ici les DLLs nécessaires au fonctionnement du contrôleur Arcus Performax 4EX :

- PerformaxCom.dll (64 bits)
- SiUSBXp.dll (64 bits)

Ces fichiers sont indispensables pour la communication USB avec le contrôleur via pylablib.

Téléchargez-les sur le site officiel Arcus Technology ou récupérez-les auprès de votre administrateur système.
